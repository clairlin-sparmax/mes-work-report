# A I S P 報工系統

此專案提供一個可在手機與平板開啟的線上報工頁面，符合需求：

1. 系統名稱為 **A I S P 報工系統**。
2. 可透過 QR Code 掃碼進入指定生產報工資料檔（用 `dataset` 參數帶入）。
3. 進入後可選擇報工人員、報工數量、開始時間與結束時間，提交報工紀錄。

## 使用方式

### 1) 本機啟動

在專案目錄執行：

```bash
python3 -m http.server 8080
```

看到 `Serving HTTP on ... port 8080` 代表服務已啟動。

### 2) 開啟系統

在瀏覽器打開：

- 一般入口：`http://localhost:8080/`
- 指定資料檔（模擬 QR Code 導向）：`http://localhost:8080/?dataset=LINE01-A`

## 如何確認「可以被執行」

請依序檢查以下項目：

1. 頁面可正常開啟，標題顯示 **A I S P 報工系統**。
2. `dataset` 參數有被帶入，頁面顯示：`生產報工資料檔：LINE01-A`。
3. 可選擇報工人員、輸入數量、開始/結束時間並按下「送出報工」。
4. 送出後，下方表格會新增一筆報工紀錄。
5. 若結束時間早於或等於開始時間，系統會提示錯誤（驗證生效）。

## QR Code 建議格式

正式上線時，可將 QR Code 指向：

`https://your-domain.example/?dataset=LINE01-A`
